# Plan 05-03 — Window + Blinds + Procedural Night-City Texture

**Phase:** 05-room-shell · **Plan:** 03 · **Wave:** 3 · **Depends on:** 05-02 (Walls + Ceiling)

## Objective

Cut a 1.2 × 1.5 m window into the back wall at offset-right (per CONTEXT D-10), mount it with horizontal aluminum venetian blinds half-open (D-12..15), and render a deterministic procedural night-city pattern behind it via a 1024 × 1024 CanvasTexture authored from a seeded PRNG (D-16..19). Affirm the OPSEC contract: zero real photos, zero identifiable skylines, no employer- or location-revealing references in the texture authoring code.

This is the last visual artifact of Phase 5. After it lands, the room reads as a fully enclosed living space and Phase 6 can place its floor anchors (server rack + bed) against the now-existing back wall.

## ROADMAP Phase 5 Success Criteria addressed

- **#3** — Procedural canvas-texture night-city behind horizontal blinds, OPSEC-clean asserted in component file header comment.

## Phase-level must_haves
